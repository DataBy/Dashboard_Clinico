You are a senior SaaS dashboard UI designer.

Extend the existing clinical dashboard layout without redesigning or moving any current components.

The dashboard already contains:
• Left sidebar navigation
• Main content cards
• A calendar on the right panel

Your task is to add two compact widgets directly BELOW the calendar to fill the empty vertical space while keeping the interface clean and minimal.

## Design Rules

Do NOT change:
- Sidebar layout
- Card styles
- Typography
- Spacing of the main layout
- Color palette
- Existing components

Only add new widgets under the calendar.

## Widget 1 — "Actividad de Hoy"

A small analytics card showing quick metrics for the current day.

Content:
• Pacientes atendidos
• En espera
• Diagnósticos registrados
• Tiempo promedio de atención

Design:
- Compact card
- Same rounded corners as other cards
- Light background
- Small icon indicators
- Clean minimal typography

## Widget 2 — "Pacientes en Espera"

A small list of the next patients in queue.

Content:
Maximum 4 patients:
• Nombre del paciente
• Indicador de prioridad (color badge)

Example:

Juan Pérez — Alta prioridad  
María Gómez — Media  
Carlos Ruiz — Baja  

Design:
- Minimal list style
- Small avatars or icons optional
- Color priority badges
- Compact vertical spacing

## Layout

Right column structure must be:

Calendar  
Actividad de Hoy  
Pacientes en Espera

Both widgets should visually match the style of the existing dashboard cards.

## Goal

Fill the empty space under the calendar with useful clinical information while maintaining a minimal, modern SaaS dashboard aesthetic.
